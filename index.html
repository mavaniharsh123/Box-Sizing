Q.1:- Explain box sizing Properties with examples?

The box-sizing property defines how the width and height of an element are calculated, should they include padding and borders, or not.

Border-box : Border-box include padding and border of an element within.

For example : if width is 300px, padding is 10px and border is 5px, then content width is 270px, but overall width (border-box) will be 300px. Means content width will reduce to maintain border-box

My Border box

Content-box : Content-box is default value of div tag. By adding border and padding to a div with fixed width can change overall width ( border box) of div.
For example: If width is 300px, padding is 10px and border is 5px, then content width is 300px, but overall width (border-box) will be 330px by adding padding and border with width.
